## Supports:
- Bitcoin - BTC
- Etherium - ETH
- Litecoin - LTC
- Doge - DOGE
- TON
- USDT TRC20
- Solana - SOL
- Steam Trade URL

## Features:
- No UAC required
- AutoRun (can repair self)
- Open Source
- 6MB .exe (with UPX, 2-3MB)
- Does not require dependencies (Go binary)
- Builder (batch)
- Send logs to Telegram Bot
- Self Copy
